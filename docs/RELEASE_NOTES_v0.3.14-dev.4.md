# GoAgent v0.3.14-dev.4

GoAgent v0.3.14-dev.4 is a development release focused on top-tier multilingual product polish. It extends the desktop UI and AI teacher language path to Simplified Chinese, Traditional Chinese, English, Japanese, Korean, Thai, and Vietnamese, and fixes remaining untranslated status and KataGo official model preset labels. This release keeps the v0.3.14-dev.3 baseline: reliable full-game KataGo analysis, real local KataGo + LLM teaching evaluation, opt-in persistent KataGo analysis engine, spawn fallback, grounded shape recognition, local pattern matching, teacher sessions, and release quality gates.
